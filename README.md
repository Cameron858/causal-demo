# Causality Demo

This repository contains examples and demonstrations of causal inference techniques applied to real-world datasets. The aim is for it to explore methods for identifying causal relationships and estimating treatment effects using observational data.

**Note**: As this is for a demo, I will be adding figures, data and notebook outputs to VCS. This is **against** common practice.
